# COREO - Stage 2 操控 × 鏡頭 × 深度 × 動態品質驗證 (Gym Prototype)

本專案為 COREO Stage 2 體感驗證房，用於 GitHub Pages 線上測試。

## 狀態聲明 (Stage 2)
- **非完整遊戲**：這是一個專注於驗證「手感、3D 深度、鏡頭捲動、回饋動畫」的 20x7 直向測試走廊。
- **視覺深度**：純 CSS 3D Transform 打造，具有透視感與牆體厚度，搭配頂部霧氣漸層。
- **動態品質 (Juice)**：實作慣性移動、撞牆微震動、Light Core 實體飛入 HUD 的粒子吸收動畫、以及無 alert() 的過關收束光效。
- **鏡頭跟隨**：由下往上推進，平滑跟隨玩家 Y 軸。
- **禁止項目已避開**：無綠色、無 alert()、無網頁捲動。
